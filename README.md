# Moustafa Magdy | Professional Portfolio

A modern, dark-themed personal portfolio website built to showcase my software engineering projects and technical skills. Designed with a focus on UI/UX, responsiveness, and clean code architecture.

## Features

- **Dark Mode UI:** Custom "Deep Void" theme with glassmorphism effects.
- **Dynamic Content:** Projects are loaded dynamically via a JSON-based API (using GitHub as a CDN).
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens using Bootstrap 5 Grid.
- **Interactive Elements:** Smooth scrolling, hover effects, and a custom code-terminal component.

## Tech Stack

- **Framework:** Angular 17 (Standalone Components)
- **Styling:** Bootstrap 5, Custom CSS, Bootstrap Icons
- **Language:** TypeScript
- **Font:** Poppins (Headings) & Inter (Body)
- **Data Source:** Custom JSON API (My JSON Server)
